# lmt-orm-enhancer
ORM增强器，针对Hibernate,Mybatis,Jdbc进行增强，简化开发，提高开发效率。

一.Hibernate的增强：

1.封装了基本Dao(IHibernateBaseDao,HibernateBaseDao)

2.增加了动态SQL的支持

二.Mybatis的增强：

1.封装了基本的Mapper(IBaseMapper)

2.通过Mybatis拦截器实现了分页查寻功能

三.Jdbc的增强：

1.封装了基本的Dao(IBaseDao,BaseDao)

2.封装了基本的Jdbc查寻工具类(JdbcUtil)

3.增加了动态SQL的支持
